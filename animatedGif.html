animated GIF

Original plant vectors in Adobe Illustrator

For the animation I decided to expand on the playfulness expressed by the group’s logo with the addition of five vector illustration representing different varieties of plants that would appear and disappear by the side of the logo typographic component in an infinite loop. This image would be suitable to be used on the group social media pages.

I originally constructed it in Adobe Illustrator using the same process described in image 6 of the image portfolio page. Later I imported all the illustrations and the logo typographic element on Photoshop’s separate layers and proceeded to add them as frames in the timeline, set the number of frame per second for each layer and position them across the timeline. Finally looped it indefinitely and saved it in GIF format.
